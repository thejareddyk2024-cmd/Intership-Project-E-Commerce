# ShopSmart-AI

ShopSmart-AI is an intelligent shopping assistant leveraging AI and RAG (Retrieval-Augmented Generation) to provide personalized recommendations and insights.

## Project Structure

- `frontend/`: User interface
- `backend/`: Core application logic and API
- `ai-service/`: Specialized AI processing
- `rag-service/`: Retrieval-Augmented Generation service
- `specs/`: Project specifications and documentation
- `docs/`: General documentation

## Getting Started

1. Clone the repository.
2. Copy `.env.example` to `.env` and configure your environment variables.
3. Refer to the `specs/` directory for detailed architecture and requirements.
